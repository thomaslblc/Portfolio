Booki

Contexte L'entreprise Booki souhaite développer un site internet qui permette aux usagers de trouver des hébergements la ville de leur choix. J'ai été chargé d'intégrer les maquettes du site en HTML et CSS. Problématiques Dans ce projet il a fallu respecter les spécifications et contraintes techniques inscrites dans la note de synthèse qui nous était fournie ainsi que la maquette. Il a fallu notamment que les cartes 'hébergement' soit cliquables dans leur entièreté. Enfin, le site devait être responsive avec 3 breakpoints donnés et le code devait être passé aux validateurs W3C. Compétences mises en œuvre HTML, CSS Lien vers le projet

Nina Carducci

Contexte Nina Carducci, photographe, possède un portfolio comprenant beaucoup d'images. Son site n'est pas bien optimisé et quelques bugs sont présents. Après audit, je lui propose d'optimiser les images, le code et la structure de son site, ainsi que de corriger les bugs. Problématiques Dans ce projet, il a fallu faire un audit LightHouse pour voir d'où provenaient les soucis d'optimisation. J'ai donc travaillé sur le redimensionnement et le formatage des images, sur la minification des fichiers JS et CSS et sur l'optimisation SEO (notamment des rich snippets). Il a également fallu travailler sur l'accessibilité en réorganisant le code et en respectant les règles de la WCAG. J'ai enfin corrigé les bugs présents dans le code Javascript. Compétences mises en œuvre HTML, CSS, JavaScript, Réaliser un audit Lighthouse, Utilisation du 'srcset' Lien vers le projet

Mon Vieux Grimoire

Contexte Une librairie souhaite ouvrir un site de notation de livres qui permettra aux utilisateurs connectés d'en ajouter un nouveau et d'y mettre une note visible par le public. J'ai été chargé de développer la partie back end du site en Node.js. Problématiques Dans ce projet, il a fallu suivre le cahier des charges fourni avec l'API. J'ai commencé par créer le serveur et le routeur grâce au framework Express JS et paramétrer la base de données MongoDB grâce à Mongoose. J'ai créé différents middlewares pour l'authentification, la gestion des requêtes et le retraitement des images avec multer et sharp (green code). Enfin grâce aux méthodes HTTP, j'ai créé différents controleurs pour manipuler la base de données. Compétences mises en œuvre Node JS, Express JS, MongoDB, Mongoose. Utilisation des librairies Multer et Sharp JS Lien vers le projet
